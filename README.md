# 🎬 AniStake

AniStake is a simple anime showcase website built using **Node.js, Express, HTML, and TailwindCSS**.
It displays popular anime titles with ratings and includes a **search feature** to filter anime cards.

---

## 🚀 Features

* 🎥 Anime card layout
* 🔎 Search anime by name
* ⚡ Fast static site using Express
* 🎨 Styled using TailwindCSS
* 📱 Responsive grid layout
* 🖥 Node.js backend server

---

## 🛠 Tech Stack

### Frontend

* HTML5
* Tailwind CSS
* JavaScript

### Backend

* Node.js
* Express.js

## 📦 Installation

Clone the repository

git clone https://github.com/yourusername/anistake.git

Move into the project folder

cd anistake

Install dependencies

npm install

---

## ▶ Run the Server

Start the server

npm start

or run in development mode

npm run dev

Server will run at:

http://localhost:5050

---

## 🔍 Search Function

The search bar filters anime cards dynamically using JavaScript.

Example:

Search: nar
Result: Naruto
